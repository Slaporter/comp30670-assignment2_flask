ReadMe.md

First need to clone repositary https://github.com/Slaporter/comp30670-assignment2.git
This can be installed using pip (pip install git+https://github.com/Slaporter/comp30670-assignment2.git)

Ensure flask is installed (pip install flask)
Once these are installed, from the command line run the application (python run.py)
The app will be run on local host 5000 (localhost:5000) in the url of any web browser.
